Module 3 - CSS Basic

Import

CSS Inline - Styles placed on the same line of HTML code.
CSS Intern - Styles placed in the same HTML file (Inside the "head" tag).
CSS External - Styles placed in the CSS file. The file is imported inside the "head" tag.

Selectors

Class - Use the "class" property in the HTML element (class="myClass"). And use a point "." to manipulate the class on the CSS file.

ID - Use the "id" property in the HTML element (id="myElement").
And use a "#" to manipulate the id element on the CSS file.

Property

Colors - By name, rgb, rgba & hexadecimal.

box-sizing - ajusta o referencial da direção da borda. Por padrão, a direção da borda de um elemento é criada para fora. Quadrado com 100px, 5px de borda, direção pra fora (padrão):
